## Run Locally
Clone the project
```bash
  git clone https://github.com/thobekabovana/Boolean-work.git
```
Go to the project directory
```bash
  cd Boolean
```
Install dependencies
```bash
  npm install
```
Start the react server
```bash
  npm run dev
```
## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
